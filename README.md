# CheeseTama: The Milkroom

**CheeseTama: The Milkroom**은 작은 치즈타마 알을 돌보고, 우유를 먹이며, 말랑한 친구로 부화시켜 가는 **PC 우선 감성 캐주얼 육성 게임**입니다.

아늑한 밀크룸에서 치즈타마의 상태를 확인하고, 우유와 간식을 챙겨주고, 함께 놀고, 방을 꾸미며 나만의 치즈타마를 성장시키는 것이 핵심입니다.

> 이 README는 공개용 기준으로 작성되었습니다. 플레이어가 직접 발견해야 하는 스포일러성 콘텐츠, 숨겨진 수집 요소, 후반부 조건은 문서에 기재하지 않습니다.

## 핵심 특징

### 치즈타마 알 돌보기

처음의 치즈타마는 작은 알 형태입니다. 플레이어는 우유를 주고, 상태를 관리하고, 쓰다듬고, 청소하며 치즈타마가 조금씩 반응하는 과정을 지켜봅니다.

### 말랑한 부화 경험

성장 조건을 만족하면 치즈타마 알이 깨지고, 말랑한 슬라임형 치즈 캐릭터가 등장합니다. 부화 이후에는 움직임, 표정, 반응이 더 풍부해집니다.

### 우유 중심 성장 시스템

치즈타마의 먹이 시스템은 일반 음식이 아니라 **우유, 발효우유, 유제품 간식**을 중심으로 구성됩니다. 어떤 우유를 얼마나 자주 먹였는지에 따라 성장 방향과 반응이 달라집니다.

### 상태 관리

치즈타마는 시간이 지나면 상태가 변합니다. 플레이어는 하루 여러 번 짧게 접속해 아래 상태를 관리합니다.

| 상태 | 설명 |
|---|---|
| 포만감 | 낮아지면 배고픔 상태가 됩니다. |
| 기분 | 낮아지면 반응과 놀이 효율이 줄어듭니다. |
| 청결도 | 낮아지면 방 청소가 필요합니다. |
| 졸림 | 높아지면 수면 관리가 필요합니다. |
| 건강 | 먹이 선택과 청결 상태에 영향을 받습니다. |
| 애정도 | 돌봄 행동과 상호작용으로 쌓입니다. |

### 먹이의 장단점

먹이는 모두 긍정 효과만 주지 않습니다. 특정 우유나 간식을 반복해서 먹이면 질림, 과포만, 나른함 같은 가벼운 패널티가 생길 수 있습니다. 단, 패널티는 회복 가능한 방식으로 설계합니다.

### 밀크룸 꾸미기

밀크룸은 치즈타마가 생활하는 메인 공간입니다. 성장과 보상에 따라 배경, 조명, 침대, 우유 보관함, 소품 등을 꾸밀 수 있습니다.

### 도감 수집

우유, 성장 기록, 이벤트, 치즈타마의 다양한 모습을 도감에 기록합니다. 도감은 수집 욕구와 장기 플레이 동기를 제공합니다.

```text
게임 접속

→ 치즈타마 반응 확인
→ 성장도 및 보상 획득
→ 도감 기록 또는 이벤트 발생
→ 다음 접속 유도
```

## 하루 플레이 구조

CheeseTama는 한 번에 오래 플레이하는 게임이 아니라, 하루에 여러 번 짧게 접속하는 다마고치식 플레이를 목표로 합니다.

| 플레이 유형 | 접속 횟수 | 1회 플레이 | 하루 총량 |
|---|---:|---:|---:|
| 가벼운 플레이 | 1~3회 | 1~3분 | 약 3~8분 |
| 일반 플레이 | 4~6회 | 1~3분 | 약 8~18분 |
| 집중 플레이 | 6~8회 | 1~3분 | 약 15~25분 |

예시 루틴:

```text
아침: 밤새 변한 상태 확인, 우유 주기
점심: 간식 또는 짧은 놀이
오후: 우유 성장도 관리, 조합 시도
저녁: 청소, 놀이, 이벤트 확인
자기 전: 수면 상태 설정
```

## 개발 원칙

- 플레이어에게 과도한 처벌을 주지 않는다.
- 치즈타마와의 애착을 최우선으로 둔다.
- 먹이 시스템은 우유 중심 정체성을 유지한다.
- 치즈 종류는 먹이 아이템으로 사용하지 않는다.
- 공개 문서에서는 스포일러성 콘텐츠를 노출하지 않는다.
- 후반부 발견 요소는 조건표가 아니라 플레이 경험으로 발견하게 한다.
- PC 우선 UX를 기준으로 만들되, 모바일 포팅 가능성을 고려한다.
