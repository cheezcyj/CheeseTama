# CheeseTama: 밀크룸

CheeseTama는 우유로 자라는 작은 치즈 생명체를 돌보는 피시 중심 육성 게임입니다. 플레이어는 밀크룸에 들어가 CheeseTama의 상태를 확인하고, 우유/조합/간식/놀이/청소/수면 행동으로 성장과 도감 기록을 쌓습니다.

현재 저장소는 최종 게임이 아니라, 유니티에서 핵심 반복 구조와 비주얼 방향을 빠르게 검증하기 위한 개발 버전입니다.

## 현재 상태

- 유니티 6 `6000.0.78f1` 기반 프로젝트
- MIT 라이선스
- 부트, 밀크룸, 도감, 개발자 씬 구성
- 로컬 저장/불러오기/초기화
- 밀크룸 체류 시간, 일일 루틴, 초기 재화, 도감 기록 프로토타입
- 기본 우유와 별빛 우유 성장 기록 분리
- 3D 카툰 렌더링 방향의 CheeseTama 캐릭터
- 따뜻한 목재 방 콘셉트의 절차적 밀크룸 디오라마
- 설정 데이터 관리, 도감/꾸미기 오버레이, 개발자 패널

## 실행 방법

1. 유니티 허브에서 `C:\Users\user\Desktop\CheeseTama` 폴더를 엽니다.
2. 유니티 상단 메뉴에서 새로고침을 실행합니다.
3. 콘솔에 빨간 컴파일 오류가 없는지 확인합니다.
4. 유니티 상단 메뉴에서 `CheeseTama > 시작 씬 빌드`를 실행합니다.
5. 밀크룸 씬을 열고 재생 버튼을 누릅니다.

## 빠른 확인

1. 밀크룸 씬에서 재생합니다.
2. 하단의 `우유`, `조합`, `간식`, `놀이`, `청소`, `수면` 버튼을 눌러 봅니다.
3. 상태 수치, 메시지 바, CheeseTama 반응이 바뀌는지 확인합니다.
4. 상단의 `도감`, `꾸미기`, `설정` 메뉴를 확인합니다.
5. 개발자 패널에서 `1시간 경과` 버튼을 눌러 시간 진행을 확인합니다.

## 빌드 검증

유니티가 프로젝트 파일을 만든 뒤에는 아래 명령으로 C# 컴파일을 확인할 수 있습니다.

```powershell
dotnet restore CheeseTama.csproj
dotnet build CheeseTama.csproj --no-restore
```
